# Hardening Phase 1 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add admission webhooks (validation + defaulting), finalizers (Release/Application/Pipeline cleanup), and error handling improvements (transient retry, rate limiting, MaxConcurrentReconciles) to make the Paprika operator production-ready.

**Architecture:** Three independent subsystems: (1) per-CRD validating+defaulting webhooks scaffolded via kubebuilder, (2) finalizer patterns using `controllerutil.AddFinalizer`/`RemoveFinalizer` in existing controllers, (3) configurable `MaxConcurrentReconciles`, QPS/Burst bumps, and error return fixes in controllers.

**Tech Stack:** kubebuilder webhook scaffolding, controller-runtime webhook server, `sigs.k8s.io/controller-runtime/pkg/controller/controllerutil`, Go `rest.Config` rate limiting

**Spec:** `docs/superpowers/specs/2026-06-10-hardening-design.md`

- [ ] **Add `cleanup` method to ReleaseReconciler**

Define a package-level `managedGVRs` slice (like the spec) with the 3 GVRs tracked for canary cleanup. This is separate from `knownGVRs` (which is a map, not a slice):

```go
var managedGVRs = []schema.GroupVersionResource{
    {Group: "apps", Version: "v1", Resource: "deployments"},
    {Group: "", Version: "v1", Resource: "services"},
    {Group: "networking.k8s.io", Version: "v1", Resource: "ingresses"},
}

func (r *ReleaseReconciler) cleanup(ctx context.Context, release *paprikav1.Release) error {
    // Delete manifest snapshot ConfigMap (stored in operator namespace by storeManifestSnapshot)
    cmName := release.Status.RenderedManifestSnapshot
    if cmName != "" {
        if err := r.Delete(ctx, &corev1.ConfigMap{ObjectMeta: metav1.ObjectMeta{
            Name:      cmName,
            Namespace: r.Namespace,  // operator namespace — same as storeManifestSnapshot
        }}); err != nil && !apierrors.IsNotFound(err) {
            return fmt.Errorf("deleting manifest snapshot: %w", err)
        }
    }

    // Delete canary resources owned by this release
    selector := labels.SelectorFromSet(labels.Set{"track": "canary", "paprika.io/release": release.Name})
    for _, gvr := range managedGVRs {
        if err := r.DynamicClient.Resource(gvr).Namespace(release.Namespace).DeleteCollection(
            ctx, metav1.DeleteOptions{}, metav1.ListOptions{LabelSelector: selector.String()},
        ); err != nil && !apierrors.IsMethodNotSupported(err) {
            return fmt.Errorf("cleaning up canary resources: %w", err)
        }
    }
    return nil
}
```

API Types

Paprika defines 6 custom resource types plus supporting configuration structs, all under the pipelines.paprika.io API group.

API Group & Version

apiVersion: pipelines.paprika.io/v1alpha1 kind: <Kind>

Resource Summary

KindDescriptionSubresource
ApplicationTop-level user-facing primitiveStatus
TemplateSource configuration (Helm/Git/S3)Status
PipelineBuild/test step workflowStatus
StageDeployment environmentStatus
ReleaseSingle promotion lifecycleStatus
ArtifactBuild output referenceStatus

Common Types

ApplicationSource

type:       string   # helm, git, s3 repoUrl:    string revision:   string path:       string chart:      ChartRef bucket:     string   # S3 key:        string   # S3 region:     string   # S3 endpoint:   string   # S3 custom endpoint secretRef:  string   # Auth secret pollInterval: string # Source polling interval

ChartRef

repo:    string   # Chart repo URL name:    string   # Chart name version: string   # Chart version path:    string   # Local chart path

GitSourceSpec

repoUrl:   string revision:  string   # Branch, tag, or commit path:      string   # Subdirectory within repo auth:      GitAuth  # SSH key or token

bucket:    string key:       string region:    string endpoint:  string auth:      S3Auth  # Access key or IRSA
